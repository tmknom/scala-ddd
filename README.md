# scala-ddd

ScalaでDDDを実践するよ！


## Requirement

* Java8
* Activator


## Usage

### コンパイル

```bash
activator compile
```

### テスト

```bash
activator test
```

### 静的解析：Scalastyle

```bash
activator scalastyle
```


## Installation

```bash
git clone git@github.com:tmknom/scala-ddd.git && cd scala-ddd
activator compile
```
